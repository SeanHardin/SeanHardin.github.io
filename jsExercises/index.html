Part 1:
var z; alert(typeof z);
displays undefined in the alert, because z was never assigned a value.

alert(!!false);
displays false, because it first negates false into true, then again back to false.

alert(7%2);
displays 1, the remainder after dividing 7 by 2.

var name = "Jim"; alert(name + " Smith");
displays Jim Smith, the concatenated string.

var myNum = "5"; alert(myNum+2);
displays 52, since the var was of type string so it concatenated instead of adding.


Part 2:
2.1 - Java code needs to be compiled before it can run, and also requires a dedicated environment to write. Javascript on the other hand can just be written in plaintext and run through any web browser
2.2 - the + operator is used for adding 2 objects together, and does different things depending on the types of objects used. with numbers it adds them together, with strings it concatenates them together instead.
The ++ operator adds 1 to the variable, if its a number it works, if a string then it breaks it and becomes a NaN.
2.3 - window.alert creates a dialog with the passed text, console.log prints the passed text to the console, document.write adds the passed text directly into the document
2.4 - the = assigns a the value on the right to the variable on the left.
the == operator checks if the value on the left matches the value on the right.
the === operator checks that the value and type match of the 2 values.
2.5 - a while loop runs until the condition given becomes false.
a do-while loop runs once regardless, then functions like a normal while loop.
